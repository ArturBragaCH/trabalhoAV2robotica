Código Arduino

Clique em Copiar código para colar no editor do Tinkercad ou no Arduino IDE.

// Exemplo: piscar LED no pino 13
const int ledPin = 13;

void setup() {
  pinMode(ledPin, OUTPUT);
}

void loop() {
  digitalWrite(ledPin, HIGH);
  delay(500);
  digitalWrite(ledPin, LOW);
  delay(500);
}
